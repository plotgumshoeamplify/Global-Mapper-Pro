# Global Mapper Pro

**Role:** desktop GIS workbench  
**Strength:** LiDAR + terrain + fast exports  

## Data you can load

- Point clouds (LAS/LAZ)
- DEMs and slope grids
- Shapefiles, GeoPackage, KML
- Satellite mosaics with georeference

## Analysis menu (practical picks)

| Tool | Output |
|------|--------|
| Contour generator | SHP/DXF elevation lines |
| Watershed | Basin polygons |
| Cut/fill | Volume reports for earthworks |
| Viewshed | Line-of-sight maps |

## Field → office flow

```
UAV/LiDAR capture → classify ground → DEM → contours → CAD/GIS handoff
```

Pro license adds automation hooks for repetitive county/municipal deliverables.
